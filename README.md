# NovaTech Hub Website

## Overview
NovaTech Hub is a simple electronics business website created as a group web development project. The website represents an online electronics store that showcases computers, smartphones, and accessories. It allows users to browse products, learn about the business, create an account interface, and contact the company.

The project demonstrates website structure, page navigation, styling with CSS, and collaborative development using GitHub.

---

## Features
- Home page introducing the business
- Products page displaying electronics
- About page describing the company
- Contact page for customer inquiries
- Login page interface
- Sign up page interface
- Clean navigation across all pages

---

## Technologies Used
- **HTML** – for website structure
- **CSS** – for styling and layout
- **Git & GitHub** – for version control and team collaboration
